1. BarChartVisual
Méthodes :  
- add_categorical_dimension_field
- add_date_dimension_field
- add_numerical_dimension_field
- add_numerical_measure_field
- add_calculated_measure_field
- add_color_*
- add_column_hierarchy
- set_bars_arrangement
- set_orientation
- set_scroll_bar_visibility
Options :
- orientation (horizontal or vertical)
- BarsArrangement (clustered, stacked…)
- Axis options (grid lines, axis line visibility, offset)
- Scrollbar visibility
Limites :
- Une seule catégorie principale
- Pas de multi-axe
2. LineChartVisual
Méthodes :
- add_date_dimension_field
- add_categorical_dimension_field
- add_numerical_measure_field
- add_color_*
- set_type (solid, smooth)
- set_scroll_bar_visibility



Options :
- Type :  solid or smooth
- Scrollbar visible or hidden
- Axis display options
Limites :
- Une seule dimension temporelle
3. PieChartVisual
Méthodes :
- add_categorical_dimension_field
- add_numerical_measure_field
- set_donut_type
Options :
- donut_type : whole, small, medium, large
Limites :
- Une seule mesure
- Pas de multicolor mappings avancés
