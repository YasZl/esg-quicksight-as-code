1. CLASSES DE PARAMÈTRES

Les classes suivantes définissent les types de paramètres utilisables dans 
une analyse QuickSight :

1.1. Parameter (classe parent)

Rôle :
- Classe de base pour tous les paramètres QuickSight

Attributs principaux :
- name : nom du paramètre
- default_value : valeur par défaut 
- custom_value : valeur utilisée si le paramètre n’est pas défini
- value_when_unset_option : 
    • RECOMMENDED_VALUE
    • NULL

Méthodes importantes :
- set_static_default_value(value)
- set_dynamic_default_value(column, dataset)
- set_value_when_unset(custom_value, option)

Utilité ESG :
- paramètre générique : secteur, région, seuil, catégorie…


1.2. DateTimeParameter

Paramètre de type date ou temps

Attributs supplémentaires :
- time_granularity : YEAR, QUARTER, MONTH, DAY…

Méthodes spécifiques :
- set_rolling_date_default_value(expression, dataset)
- set_time_granularity(time_granularity)

Idées ESG :
- année de reporting
- période temporelle (année, mois)


1.3. IntegerParameter

Paramètre numérique entier

Attributs :
- parameter_value_type : SINGLE_VALUED ou MULTI_VALUED

Idées ESG : année (en entier)...



1.4. DecimalParameter

Paramètre numérique décimal

Attributs :
- parameter_value_type : SINGLE_VALUED ou MULTI_VALUED



1.5. StringParameter

Paramètre textuel (catégorie, texte)

Attributs :
- parameter_value_type : SINGLE_VALUED ou MULTI_VALUED

Idées ESG :
- secteur, région ...


========================================================

2. PARAMETER CONTROLS 

Les contrôles déterminent la manière dont l’utilisateur choisit la valeur 
d’un paramètre sur une feuille QuickSight

Tous héritent de la classe parent : ParameterControl

Attributs communs :
- id : identifiant unique du contrôle
- source_parameter_name : nom du paramètre associé
- title : texte affiché dans quicksight
- options de style (facultatives) :
    • custom_label
    • font_color
    • font_decoration
    • font_size
    • font_style
    • font_weight
    • title_options_visibility

Méthode utile :
- set_title_font(color, decoration, size, style, weight)


2.1. ParameterDateTimePickerControl

Rôle :
- Contrôle pour sélectionner une date/heure

Attributs :
- date_time_format : format d'affichage

Options modifiables :
- label personnalisé
- couleur police
- taille police
- bold/italic
- visibilité du titre

Idée ESG : choisir une année de reporting 

2.2. ParameterDropDownControl

Rôle :
- Liste déroulante 

Attributs clés :
- select_all_options_visibility
- type (statique / lié à dataset)
- column_name
- data_set_identifier
- values (valeurs statiques)

Options modifiables :
- titre (font, couleur, label)
- affichage de “Select all”

Idée ESG : sélectionner un secteur, une région…


2.3. ParameterListControl

Rôle :
- Liste avec une multi-sélection + barre de recherche

Attributs :
- search_options_visibility
- select_all_options_visibility
- column_name / dataset identifier
- values (liste statique)
- type (single ou multi)

Idée ESG :choisir plusieurs secteurs, pays, entreprises…


2.4. ParameterSliderControl

Rôle :
- Slider numérique (min, max, step)

Attributs :
- minimum_value
- maximum_value
- step_size

Options :
- style du titre (police, couleur)

Idée ESG : choisir un seuil d’intensité 


2.5. ParameterTextAreaControl

Rôle :
- Zone de texte multi-ligne

Attributs :
- delimiter : séparateur entre valeurs
- placeholder visibility


2.6. ParameterTextFieldControl

Rôle :
- Champ texte simple (une ligne)

Attributs :
- placeholder visibility

Idée ESG : entrer un nom d’entreprise
